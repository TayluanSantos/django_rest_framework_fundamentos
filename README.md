# Django Rest Framework Fundamentos

## Introdução
O presente repositório tem como intuito servir como um laboratório para explorar os principais fundamentos do Django Rest Framework.Todo código desenvolvido tem como finalidade o estudo da ferramenta, com possíveis aprimoramentos de acordo com o avançar do conhecimento adquirido.

## Projeto
O projeto consiste em um CRUD de Produtos, implementando as principais funcionalidades:
- Registrar novo produto
- Listar todos os produtos
- Atualizar produto com base no ID
- Deletar produto com base no ID
